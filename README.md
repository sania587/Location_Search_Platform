## 🔍 Overview
The **Location Search Platform** is a modern mapping solution that allows users to find places across the world with precision. Built on the powerful combination of **React** and **Leaflet**, it provides a smooth, tile-based interactive map interface coupled with a fast, intuitive search experience.

---

## ✨ Key Features
- **🌍 Interactive Map**: Fluid zooming, panning, and tile loading powered by Leaflet.
- **🔍 Smart Search**: Quickly find any city, landmark, or address using the integrated search engine.
- **📍 Dynamic Markers**: Visual indicators for searched locations with informational popups.
- **🎨 Modern UI**: Clean and minimal interface built with React components and Twin.macro.
- **⚡ Lightning Fast**: Bundled with Vite for instant server start and optimized production builds.

## 🚀 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/sania587/Location_Search_Platform.git
   cd Location_Search_Platform
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Locally**
   ```bash
   npm run dev
   ```
   *Open `http://localhost:5173` in your browser.*

4. **Build for Production**
   ```bash
   npm run build
   ```

## ☁️ Deployment
This project is optimized for deployment on **Vercel**.

### One-Click Deploy
1. Install Vercel CLI: `npm install -g vercel`
2. Deploy: `vercel --prod`
